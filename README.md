# Go Workshop

1. Clone this repository

   ```bash
   go get github.com/toefel18/golangworkshop
   ```

2. Cd to each directory, read the comments how to implement the exercise and run:  

   ```go
   cd 01-print-your-name
   go run main.go
   ```
